# LeetCode 刷题目录

这部分把 C++ 基础知识点和 LeetCode 常用模板放到一起，适合作为刷题前复习和做题时速查。

如果你已经看过 `C++ 基础` 里的前 11 篇，这里会更像“面向刷题”的整理版本；如果还没系统学过 C++，建议先从基础章节回补。

## 学习路线

- [01-C++ 与 LeetCode 知识点](leetcode_notes.md)
- [02-LeetCode 常用 C++ 模板](02-常用代码模板.md)

## 怎么配合主教程使用

- 语法不熟时，优先回看 `C++ 基础` 里的变量、函数、数组字符串、指针引用。
- 开始刷题后，把这里当成速查表，重点看 STL、DFS/BFS、二分、滑动窗口、树和链表模板。
- 面试前复习时，先看知识点页，再看代码模板页，会比从零翻基础教程更快。
