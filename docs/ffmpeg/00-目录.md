# FFmpeg 文档目录（命令行 + vcpkg + libav 入门）

FFmpeg 的使用大致分两类：

- **把它当工具**：用 `ffmpeg/ffprobe` 命令处理音视频（最常见，也最省心）
- **把它当库**：用 `libavformat/libavcodec/libswscale/libswresample` 在 C/C++ 里做解码/编码/转封装

本目录按“先能用 → 再工程化 → 再深入”的顺序组织，默认面向 Windows + Visual Studio + vcpkg。
